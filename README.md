# SLC-AS-MediaOps.LIVE-Tutorial-IP_Matrix

Complete [tutorial implementation](https://docs.dataminer.services/solutions/standard_solutions/MediaOps%20Live/Tutorials/IP%20Matrix/IP%20Matrix_ConnectionHandlerScript.html) for the integration of the IP Matrix connector in the MediaOps Live solution. This package contains a connection handler automation script that detects and registers connections between source and destination endpoints based on multicast IP metadata.

## Projects

| Project | Description |
|---|---|
| [IPMatrix_ConnectionHandler](IPMatrix_ConnectionHandler/README.md) | Automation script that acts as a connection handler for the IP Matrix connector in MediaOps Live. |
| [Tutorial-IPMatrix-Package](Tutorial-IPMatrix-Package/README.md) | DataMiner application package project that bundles the solution into a `.dmapp` for deployment. |
